# 🤖 Laptop Control Bot - Website

This is the official website for the Laptop Control Bot Discord application. A modern, vibrant, and fully responsive website showcasing the bot's features and providing easy access to documentation.

## 📁 Project Structure

```
bot-website/
├── index.html              # Landing page with hero section and overview
├── css/
│   ├── styles.css         # Main stylesheet with vibrant color scheme
│   └── pages.css          # Additional styles for feature & setup pages
├── js/
│   └── script.js          # Interactive features and bot status checker
├── pages/
│   ├── features.html      # Complete list of all 25+ commands
│   ├── setup.html         # Step-by-step setup and installation guide
│   └── contact.html       # Contact information and support
├── assets/
│   └── images/            # Images and media (currently unused, placeholder)
└── README.md              # This file
```

## 🚀 Getting Started

### Option 1: Local Development
1. Clone the bot-website directory
2. Open `index.html` in your browser
3. Make changes and test locally

### Option 2: Deploy to GitHub Pages
1. Create a GitHub repository (e.g., `bot-website`)
2. Push this folder to the `main` branch
3. Enable GitHub Pages in repository settings
4. Access your site at `https://yourusername.github.io/bot-website`

### Option 3: Deploy to Vercel
1. Push to GitHub
2. Import the repository in [Vercel](https://vercel.com)
3. Automatically deployed with live preview URLs

## 🔧 Technical Details

### CSS Architecture
- **CSS Variables**: Easy theming with --primary, --secondary, --accent colors
- **Responsive Grid**: Auto-adapting layouts with minmax()
- **Animations**: Smooth transitions and keyframe animations
- **Dark Mode**: Optimized for dark theme viewing

### JavaScript Features
- **Bot Status Checker**: Async function to check if bot is online
- **Smooth Scrolling**: Navigation links with smooth scroll behavior
- **Intersection Observer**: Fade-in animations as elements scroll into view
- **Local Storage**: Cache bot status between page loads

### Browser Support
- Chrome/Edge 90+
- Firefox 88+
- Safari 14+
- Mobile browsers (iOS Safari, Chrome Mobile)

## 🔐 Security Notes

- All external links open in new tabs (`target="_blank"`)
- No sensitive data stored or transmitted
- Static HTML/CSS/JS - no backend required
- Bot token never exposed to frontend

## 📱 Mobile Responsive

The website is fully responsive with breakpoints at:
- **Desktop**: 1200px+ (full layout)
- **Tablet**: 768px-1199px (adjusted spacing)
- **Mobile**: Below 768px (single column layout)
